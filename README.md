# 💳 Credit Card Financial Dashboard (EXCEL + SQL + Power BI)

## 📊 Project Overview
The **Credit Card Financial Dashboard** is an end-to-end data analytics project built using **Microsoft Power BI** and **SQL**.  
The project focuses on analyzing credit card transactions and customer data to generate meaningful business insights through interactive dashboards.

This dashboard helps in tracking **revenue trends, transaction behavior, customer demographics, and financial performance metrics** on a weekly and quarterly basis.

---

## 🎯 Project Objectives
- Analyze credit card financial performance using structured SQL data
- Track key KPIs such as **Revenue, Interest Earned, Income, and Transaction Count**
- Understand customer behavior based on **gender, age group, card category, and income level**
- Create an interactive and visually appealing Power BI dashboard
- Demonstrate real-world **Business Intelligence & Data Analytics** skills

---

## 🛠️ Tools & Technologies Used
- **Microsoft Power BI** – Data visualization & dashboard creation  
- **SQL** – Data storage, table creation, and data preparation  
- **DAX (Data Analysis Expressions)** – Measures & calculated columns  
- **CSV Files** – Source datasets  

---

## 📌 Dashboard Features
### 🔹 Key Performance Indicators (KPIs)
- Total Revenue
- Total Transaction Amount
- Total Interest Earned
- Customer Income
- Customer Satisfaction Score (CSS)

### 🔹 Interactive Visuals
- Quarterly Revenue & Transaction Count
- Weekly Revenue Trends
- Revenue by Expenditure Type
- Gender-wise and Card Category-wise analysis
- Customer segmentation based on income level

### 🔹 Filters & Slicers
- Quarter (Q1–Q4)
- Gender (Male / Female)
- Card Category (Silver, Gold, Platinum, Blue)
- Income Level (Low / Mid / High)
- Week Start Date

---

## 📈 Business Insights Enabled
- Identify revenue growth and decline patterns over time
- Analyze spending behavior across different customer segments
- Compare transaction trends by card category
- Understand which expenditure types contribute the most revenue
- Support data-driven decision-making for financial analysis
